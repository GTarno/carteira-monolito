# Imagens do Carrossel de Parcelamentos

Adicione as seguintes imagens neste diretório para que o carrossel funcione corretamente:

## Imagens necessárias:
- `phone.png` - Imagem de um smartphone (por exemplo, iPhone)
- `car.png` - Imagem de um carro
- `travel.png` - Imagem relacionada a viagem (avião, mala, etc.)
- `laptop.png` - Imagem de um laptop/notebook
- `credit.png` - Imagem relacionada a cartão de crédito

## Recomendações:
- Formato: PNG ou JPEG
- Tamanho recomendado: 400x300px ou 16:9
- Qualidade: HD, mas otimizada para web
- Estilo: Preferencialmente com fundo limpo ou transparente

## Fontes sugeridas:
- Unsplash.com
- Pexels.com
- Freepik (com conta Premium)
- Ícones do Material Design ou similar para consistência visual
